Detailed Description

API specific to caching access.

The idea is accessing any sort of resource routes through here, and the cache layer will handle the loading. Colors and fonts must be created in styles.yoyo, (or manually before usage) and can be indexed by name. Fonts must be declared at a specific size in styles.yoyo.
