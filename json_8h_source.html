26/*
27 json wrapper
28
29 This is a light wrapper around the jansson library that provides a more convenient interface as well
30 as validation and error checking. If you try to access a value that does not exist you will recieve
31 false, with the output ptr not being modified as well as a ye_logf warning.
32
33 Functions take in the json_t * to act upon, a key to index a value via, and an output ptr to assign
34 the value to. If the key is not found, the function will return false and the output ptr will be
35 NULL. If the key is found, the function will return true and the output ptr will be assigned the
36 value.
37
38 Some reccomended convention:
39 name any root json_t objects in UPPERCASE, and ensure to json_decref them when you are done with them
40*/
41
